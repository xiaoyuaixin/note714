1、Java中的注释有三种 
	1) 单行注释:  它以  // 开头，后跟注释内容。
	2) 多行注释:  它以  /* 开头，以 */ 结尾，中间可以书写多行注释内容。
	3) 文档注释:  它以  /** 开头， 以 */ 结尾，中间可以书写多行注释内容。
	
2、注释的作用是什么？
	它用来对代码进行注解说明，告诉读者代码的含义或作用是什么。   

3、注释要注意什么？
	1) 单行注释的内容只能在一行上。
	2) 多行注释或文档注释不能嵌套使用。
	
4、什么是标识符？
	它以字母或下划线或美元符开头，后跟字母或下划线或美元符或数字组成的字符系列，称之为标识符。
	例如:  a3   _31  $3  ( a#3  a-3  )
	
5、标识符的作用是什么？
	它的作用是: 作为一种命名的规则来使用。
		即: 作为 为变量、方法、类命名时的规则。 
		也就是说: 变量名、方法名或类名等名称必须符合标识符定义。 否则，就会出错。
	
6、什么是关键字？
	在Java中，将某些单词或符号保留下来并赋予一种特定含义，作为特殊的用途。
	这些单词或符号就称之为关键字。 
	例如:  看到 class 表明生成一个类。因此，class就是一个关键字
		再如:  public   int  double  static  void 
		
7、什么是变量？
	变量是一个内存中的存储单元，是一个容器，用来存放指定类型的数据。
	但，它在使用中可以变存储的量。
	
8、变量有三个要素？
	名称	--->> 必须符合标识符的定义。
	类型    --->> 规定变量存储什么种类的数据。
	作用域  --->> 规定变量在什么范围里面使用。
	
9、变量必须先声明，后使用。
	根据变量声明的位置不同，可以分为两种:
	1) 局部变量:  在方法中或语句块中定义的变量称之为局部变量。
	
	2) 成员变量（实例变量）: 在方法外，在类的类体中声明的变量。
	
10、算术运算符:
	+  - 	* /   %  
	
	+ 有三种身份
	1) 加法, 如:  a + b;
	2) 连接, 如:  "123" + 300;   当+的左右两边有一个字符串时，则做连接运算。
	3) 正号, 如:  +a
	
	- 有两种身份
	1) 减法, 如:  a - b 
	2) 负号, 如:  -a 
	
	* 有一种身份
	乘法，如:  a * b;
	
	/ 有一种身份
	除法， 如:  a / b 
	
	注意: 当 / 的左右两边均为整数时，则结果为整数( 商 )
	
	% 有一种身份
	取模（取余），如:  a%b
	规则: 取 a / b 的余数。
